Inside Molly it's possible to style out, pimp out, relax out everything.

Themes should be installed through the backend. It will make sure that all essentials are present.

If you want to create your own theme, it's advised to start with a copy of the default include theme. It automagically
contains all the currently installed modules' templates.

Important files will be rebuild to their previous contents.
